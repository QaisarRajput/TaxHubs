# Brand Assets

Add your branding files in this folder using these exact names:

- `logo.png` (light theme header logo)
- `logo-dark.png` (dark theme header logo)
- `banner.svg` (hero/banner background + OG/Twitter preview image)

Recommended sizes:

- `logo.png` and `logo-dark.png`: around 320x96 (or similar wide logo ratio)
- `banner.svg`: 1200x630 viewBox (vector, scales without pixelation)

These files are referenced by:

- Header logo component
- Hero banner background image
- SEO metadata (`icon`, Open Graph, Twitter)
